## 代码调试方法

进入题解所在目录执行以下命令：

```bash
# C++
g++ solution.cpp -o solution -std=c++17 && ./solution

# Go
go run solution.go

# Python
python solution.py
```

每通过一个用例会输出：`PASS: CASE XXX`，未通过会输出：`FAIL: CASE XXX`。
